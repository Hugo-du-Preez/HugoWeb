Portfolio Analytics Dashboard

Business intelligence solution with embedded Power BI and Excel integration

Project: 01 / 03
Duration: 3 months
Status: Completed

🎬 Demo

Interactive walkthrough showing Power BI embedded reports and Excel integration.

🎯 Project Overview

Developed a modern web dashboard featuring embedded live Power BI reports and interactive Excel data from OneDrive, deployed on Azure Static Web Apps Free Tier. Built with vanilla JavaScript for optimal performance, demonstrating practical implementation of business intelligence concepts and cloud deployment strategies.

🏗️ Architecture Highlights

  • Azure Static Web Apps for global CDN distribution and automatic scaling
  • Power BI Service integration with public embed URLs for interactive dashboards
  • OneDrive Excel Online embedding for real-time spreadsheet synchronization
  • Zero API configuration required - public URL-based architecture
  • 100 GB monthly bandwidth with SSL certificates included
  • Responsive design optimized for desktop, tablet, and mobile devices

📊 Results & Features

  • Live Power BI reports with interactive filtering and real-time data updates
  • Excel Online integration from OneDrive with automatic synchronization
  • Zero-cost hosting on Azure Static Web Apps Free Tier (100 GB bandwidth)
  • No API configuration required - public URL-based embedding
  • Responsive design working across desktop, tablet, and mobile devices
  • Static site deployment with global CDN distribution
